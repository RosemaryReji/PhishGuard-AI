# PhishGuard AI – UI/UX Design Language Document

## Design Vision

PhishGuard AI follows a futuristic AI-assistant inspired design language that combines:
- Cybersecurity aesthetics
- AI-powered interaction
- Modern chatbot UI patterns
- Accessibility-focused UX
- Clean futuristic visuals
